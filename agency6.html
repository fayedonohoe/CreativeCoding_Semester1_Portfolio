Generative Agency - Handwriting Simulation

Creating a Handwriting Effect using Font Paths in P5.js
April 2021

Simulating Handwritten Script with Type Agents and Font Paths

The aim for this project was to replicate handwriting using generative agency in P5.
An iterative approach was taken to this development, and a breakdown of each version can be found here:
https://projects.invisionapp.com/boards/BS428NZFA3C/


Breaking Down Text
A library called opentype was used to load the chosen font, and the library g was then used to generate a path.
This allowed me to extract the commands from the string applied, and use the x and y values as position vectors for objects.

Using Two Objects: Brush and Eraser
Initially, two separate objects were used as a brush and an eraser. They were passed the Path location points on update so as they re-rendered, it appeared to be writing across the screen. While this successfully created the initial goal, I knew there was so much more I could do with it.

Using Multiple Objects: One Per Path Point
I recreated the first set of sketches, but created a new object for each path point. While this is CPU intensive, it allows for much greater flexibility and creative maneuvers.
Having access to individual points allowed me to apply various forces such as gravity and visual changes.
