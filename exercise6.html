Mini Project - Interactive Colour Tool

Creating a Colour Gradient using HSB and Loops
November 2020

Interactive Colour Tool
We were instructed to use what we have learned so far to create "something that allows a user to play with colour".
I created a colour spotlight.

This sketch creates a colour grid using the HSB as variables in a loop.
From this, a triangle fan is generated depending on the users mouse position, using similar calculations to create the spotlight shape.

I used rotation and map to give the effect of the spotlight rotating and moving around the canvas area.

The current vertex data is stored as a vector in a shape array. When the user clicks their mouse, the current vertices are pushed, as a single shape array, to a larger array called 'shapes[]' .
In the draw() function, the shapes[] array is iterated through, drawing each of the stored spotlight shapes with randomly generated center points to create a strobe effect.
