Iteration 1 - Create a Colour Gradient using HSB and Loops
After a refresher on P5, the class was instructed to create a visual gradient similar to a colour picker. This was done by creating a nested loop of squares.
The colour mode was defined to use the width and height of the canvas as paramaters.
`colorMode(HSB, width, height, 100);`
These squares were then drawn incrementally, taking the fill colour of the respective Hue and Saturation.
This created the gradient of Hue across the X axis, and Saturation on the Y axis.

Iteration 2 - Adding Save Canvas and Interactivity to The Colour Gradient
The function saveCanvas() was included below draw(). This built in P5 function allows an image of the canvas to be downloaded. We used 'timestamp' from the generative-design-library as the naming convention.
Continuing from the previous exercise, we used mouseX and mouseY in the declaration of the "step" variables to allow user interactivity.
