Not So Random Walker
A step up from random walker, an object moved with Directional Intelligence.
Appropriate angles are pre-calculated and stored in an array. The object moves in the X and Y direction by recalculating as follows:
xPos += cos(radians(angle) * stepSize);
yPos += sin(radians(angle) * stepSize);
When the object hits the edge of the canvas, it is deflected away at one of the pre calculated angles
angle = deflectAngle(xPos, yPos, angles, positions);
