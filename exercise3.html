Creating a Colour Wheel using TRIANGLE_FAN
23-09-20

Iteration 1 - Fixed Point Colour Wheel Using TRIANGLE_FAN
This exercise involved the use of the built in P5 feature, TRIANGLE_FAN.
Using this shape tool and Pythagoras' Theorem, it was possible to create a loop iterating through each vertex of the shape, positioning the outer points around in a uniform circle.
The fill colour of each segment was determinded using the coordinates of Alpha, which is the index of the loop. As the loop iterates, Alpha is incremented by the predefined stepAngle.

Iteration 2 - MouseX and MouseY Centre Points
